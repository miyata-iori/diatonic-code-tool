# ダイアトニックコード進行支援ツール

## 制作期間

約3週間

## 概要

キーとメジャー/マイナーを選択するとそのキーと調性に対応したダイアトニックコードが表示されます。
表示されたダイアトニックコードをクリックすると、
おすすめのコード進行とそのおすすめ理由を表示するWebアプリです。

---

## 使用技術

- HTML
- CSS
- JavaScript

---

## 主な機能

- キー変更
- Major / Minor切り替え
- ダイアトニックコード表示
- おすすめコード表示
- コードトーンの表示
- コードタイプごとの色分け

---

## 工夫した点

- JavaScriptでダイアトニックコードを動的に生成し、キーや調性が変わっても同じ処理を再利用できるようにしました。
- datasetを利用し、コード情報をクリックしたときに簡単に呼び出せるようにしています
- おすすめのコード進行ルールをオブジェクトとして管理し、データを書き換えるだけでおすすめ進行のルールを変更できるようにしました。
- コードタイプ（Major・Minor・Diminished）ごとに色分けし、視覚的にも判別しやすいUIを意識しました。

---

## 今後追加したい機能
- クリックしたコードが実際にどんな音なのかを再生する機能
- コードの選択、非選択ボタンを実装し、そのコード進行を通しで演奏する機能
- 選択したコードの組み合わせをセーブしておく機能
- 選択したコードのコードトーンを鍵盤などで視覚的に見えるようにする機能
- コードの転回形に対応する機能
- ジャンルに応じてコードのおすすめ度をつける機能
- 7thコードなどのダイアトニックコード以外への対応
